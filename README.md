homebrew-iotexproject
==============
Homebrew Tap for IoTex 

## Installation
```
brew tap iotexproject/ioctl
```
### ioctl
To install the latest 1.1 release:
```
brew install ioctl
```
Run to check your installation:
```
ioctl-darwin-amd64 version
```
Set Alias:
```
alias ioctl='ioctl-darwin-amd64'
```
